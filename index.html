Flight passenger number prediction using Machine Learning

A supervised learning problem given as a project in the "Data Mining in Dtabases and World Wide Web" course in Computer Science Department of AUEB. It is a classification problem that asks to determine the 'PAX' variable for a set of flights that happened in U.S.A. Each flight contains some features (Arrival / Departure airport, date etc.) and for some flights 'PAX' variable is already determined (so we use them as our training dataset). We have to keep in mind that 'PAX' (passengers approximately) variable is strongly related to the number of passengers a flight is carrying. The goal of the project is to predict the value of 'PAX' variable for the flights that it is not already predicted.
